# 摘要

随着人工智能和自然语言处理技术的飞速发展，对话式信息检索系统逐渐成为智能应用中的重要组成部分 [43][59]。这类系统能够与用户进行多轮对话，理解并生成自然语言回答，广泛应用于智能客服、个人助手、医疗咨询、教育辅导等领域。近年来，在大语言模型（Large Language Models, LLMs）的推动下，对话式信息检索系统的性能得到了显著提升 [8][31]。然而，当前的系统依然面临诸多挑战，包括对话历史中的关键信息遗漏、幻觉问题、个性化知识利用不足以及检索准确性与效率的平衡等 [34][35][36][37]。

针对上述问题，本文做出了如下研究和创新：

**（1）基于个人文本知识库的个性化检索方法**。为了满足用户个性化的信息需求，本文提出了一种利用个人文本知识库（Personal Textual Knowledge Base, PTKB）进行个性化对话信息检索的方法 [2]。该方法通过大语言模型从 PTKB 中选择与当前查询相关的知识，并生成个性化的查询改写。针对 PTKB 噪声问题，提出了基于相似度粗筛和 LLM 精筛的混合选择策略。实验结果表明，在提供高质量引导的情况下，大语言模型能够生成更合适的个性化查询，用户满意度达到 4.5 分（5 分制），显著优于非个性化方法 [42][69]。

**（2）基于注意力机制的关键词提取方法**。为了解决对话检索中大语言模型查询重构时的关键信息遗漏和幻觉问题，本文提出了一种基于注意力机制的关键词提取框架 [1]。该方法利用大语言模型在生成重构查询时的注意力分布，识别并提取对话历史中的显著关键词，同时无需引入额外的计算开销。通过注意力集中机制和输出 token 选择，该方法在多个对话检索数据集上显著优于传统的查询改写方法，MRR 指标达到 0.623，接近人工改写水平（0.651）[5][6][7]。

**（3）上下文压缩与生成优化策略**。为了在生成阶段处理长上下文输入，本文提出了一种上下文压缩策略 [37]。该方法在保持生成质量的同时，有效压缩了对话上下文，50% 压缩率下生成质量下降仅 3.5%，而推理速度提升约 40%。此外，通过门控演示学习，帮助模型理解任务并指导回复生成，减弱了示例不恰当时对模型生成的负面影响 [9][15]。

**（4）可解释性分析与系统实现**。对大语言模型的内部机制进行了深入的可解释性分析 [32]，研究了不同注意力层和注意力头对关键词选择和信息检索任务的贡献。发现中间层（6-8 层）和语义头的关键词提取效果最好。基于上述研究，设计并实现了一个完整的对话式信息检索系统，响应时间 P50 为 320ms，单实例吞吐量达 50 QPS，在多个实际应用场景中展现了良好的性能 [59]。

本文的研究成果为对话式信息检索系统提供了一种实用且有效的解决方案，在智能客服、个人助手、教育辅导等实际应用场景中展现了良好的性能。主要创新点包括：提出了基于注意力机制的关键词提取方法，无需额外训练即可利用 LLM 的注意力权重识别关键词；提出了基于 PTKB 的个性化检索框架，通过混合知识选择策略有效利用了用户的个人知识；设计了上下文压缩策略，在保持信息完整性的同时提升了系统效率；实现了完整的对话式信息检索系统，在实际场景中验证了方法的有效性。

**关键词**：对话式信息检索；检索增强生成；注意力机制；个性化检索；大语言模型；查询改写；个人文本知识库
