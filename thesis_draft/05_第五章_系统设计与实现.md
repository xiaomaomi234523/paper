# 第五章 系统设计与实现

## 5.1 系统总体架构

基于前述研究，本文设计并实现了一个完整的对话式信息检索系统。系统采用模块化架构，如图 5-1 所示（图略）。

### 5.1.1 架构层次

系统分为以下层次：

**（1）用户接口层**
- Web 界面：基于 React 的交互式界面
- API 接口：RESTful API 支持第三方集成
- 消息队列：处理并发请求

**（2）核心服务层**
- 对话管理：维护对话状态和历史
- 查询处理：关键词提取和查询改写
- 检索服务：文档检索和排序
- 生成服务：答案生成和优化

**（3）数据存储层**
- 文档索引：Elasticsearch 存储和索引文档
- 向量数据库：存储文档和查询的向量表示
- 用户数据库：存储用户信息和 PTKB

**（4）基础设施层**
- 模型服务：LLM 推理服务
- 缓存服务：Redis 缓存热点数据
- 日志监控：系统监控和日志记录

### 5.1.2 数据流

系统的典型数据流如下：

1. 用户输入查询和对话历史
2. 对话管理模块更新对话状态
3. 查询处理模块进行关键词提取和查询改写
4. 检索服务使用改写后的查询检索文档
5. 生成服务基于检索结果生成回答
6. 返回回答并更新对话历史

## 5.2 核心模块实现

### 5.2.1 关键词提取模块

**输入**：对话历史 $H$
**输出**：关键词列表 $K$

**实现步骤**：

```python
class KeywordExtractor:
    def __init__(self, llm_model, top_k=5):
        self.llm = llm_model
        self.top_k = top_k
    
    def extract(self, dialogue_history):
        # 1. 生成查询改写并获取注意力权重
        response = self.llm.generate(
            input=dialogue_history,
            output_attention=True
        )
        
        # 2. 提取注意力权重
        attention_weights = response.attention_weights
        
        # 3. 计算注意力集中得分
        acs_scores = self._calculate_acs(attention_weights)
        
        # 4. 选择 Top-K 关键词
        keywords = self._select_top_k(acs_scores, self.top_k)
        
        return keywords
    
    def _calculate_acs(self, attention_weights):
        # 计算每个输入 token 的平均注意力权重
        acs = np.mean(attention_weights, axis=0)
        return acs
    
    def _select_top_k(self, acs_scores, k):
        # 选择 ACS 最高的 K 个 token
        top_indices = np.argsort(acs_scores)[-k:][::-1]
        keywords = [self.tokens[i] for i in top_indices]
        return keywords
```

**优化策略**：
- **批量处理**：同时处理多个对话，提升吞吐量
- **缓存机制**：缓存常见对话的关键词
- **异步处理**：关键词提取与检索并行执行

### 5.2.3 检索模块

**输入**：查询 $q$
**输出**：排序后的文档列表 $D_{ranked}$

**实现**：

```python
class HybridRetriever:
    def __init__(self, sparse_retriever, dense_retriever):
        self.sparse = sparse_retriever  # BM25
        self.dense = dense_retriever    # DPR
    
    def retrieve(self, query, top_k=100):
        # 1. 稀疏检索
        sparse_results = self.sparse.search(query, top_k=top_k)
        
        # 2. 稠密检索
        dense_results = self.dense.search(query, top_k=top_k)
        
        # 3. 融合结果（RRF）
        fused_results = self._reciprocal_rank_fusion(
            sparse_results, dense_results
        )
        
        return fused_results[:top_k]
    
    def _reciprocal_rank_fusion(self, results1, results2, k=60):
        # 倒数排名融合
        scores = {}
        for rank, doc in enumerate(results1):
            scores[doc.id] = scores.get(doc.id, 0) + 1 / (k + rank)
        for rank, doc in enumerate(results2):
            scores[doc.id] = scores.get(doc.id, 0) + 1 / (k + rank)
        
        ranked_docs = sorted(scores.items(), key=lambda x: x[1], reverse=True)
        return [doc for doc_id, score in ranked_docs]
```

```python
class AnswerGenerator:
    def __init__(self, llm_model, context_compressor):
        self.llm = llm_model
        self.compressor = context_compressor
    
    def generate(self, query, documents):
        # 1. 上下文压缩
        compressed_context = self.compressor.compress(
            documents=documents,
            max_tokens=2000
        )
        
        # 2. 构建提示
        prompt = self._build_prompt(query, compressed_context)
        
        # 3. 生成回答
        answer = self.llm.generate(prompt)
        
        return answer
    
    def _build_prompt(self, query, context):
        return f"""
基于以下信息回答问题：

{context}

问题：{query}

回答：
"""
```

## 5.3 上下文压缩实现

### 5.3.1 压缩算法

```python
class ContextCompressor:
    def __init__(self, keyword_extractor, window_size=50):
        self.keyword_extractor = keyword_extractor
        self.window_size = window_size
    
    def compress(self, documents, keywords, max_tokens=2000):
        # 1. 提取关键词
        if keywords is None:
            keywords = self.keyword_extractor.extract(documents)
        
        # 2. 为每个关键词保留上下文窗口
        compressed_parts = []
        for keyword in keywords:
            context = self._extract_context(
                documents=documents,
                keyword=keyword,
                window_size=self.window_size
            )
            compressed_parts.append(context)
        
        # 3. 合并和截断
        compressed_text = ' '.join(compressed_parts)
        if len(compressed_text.split()) > max_tokens:
            compressed_text = ' '.join(
                compressed_text.split()[:max_tokens]
            )
        
        return compressed_text
    
    def _extract_context(self, documents, keyword, window_size):
        # 找到关键词位置并提取上下文窗口
        for doc in documents:
            if keyword in doc.text:
                idx = doc.text.find(keyword)
                start = max(0, idx - window_size)
                end = min(len(doc.text), idx + window_size)
                return doc.text[start:end]
        return ""
```
